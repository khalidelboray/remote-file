# SYNOPSIS

Fetch remote file using http.


# INSTALL

    $ panda install Sparrowdo::RemoteFile

# USAGE

    $ cat sparrowfile

    module_run 'RemoteFile' %(
        url => 'http://somehost/somefile.tar.gz'
    );

# Parameters

## url

Remote file url
